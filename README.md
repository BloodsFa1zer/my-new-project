# Credit System

Консольний додаток для формування пропозицій клієнту по цільових кредитах різних банків.

## Вимоги

- Java 11 або вище
- Maven 3.6+

## Збірка та запуск

```bash
mvn clean compile
mvn exec:java -Dexec.mainClass="credit.Application"
```

### Режими роботи

При запуску додатку вам буде запропоновано вибрати режим:

1. **Interactive Mode (Інтерактивний режим)** - дозволяє вам вручну досліджувати всі функції системи:
   - Пошук найкращих пропозицій кредитів
   - Вибір оптимального кредиту з урахуванням переваг
   - Перегляд всіх доступних кредитів та банків
   - Фільтрація за типом кредиту, банком, процентною ставкою
   - Пошук гнучких кредитів (з можливістю дострокового погашення)

2. **Demo Mode (Демонстраційний режим)** - автоматична демонстрація всіх можливостей системи:
   - Показує всі банки та кредити
   - Демонструє пошук та вибір кредитів
   - Показує різні типи кредитів та їх особливості
   - Відображає розрахунки та статистику
   - Ідеально для презентації вчителю!

**Рекомендація:** Для демонстрації вчителю виберіть **Demo Mode (2)** - він автоматично покаже всі можливості системи.

## Запуск тестів

```bash
mvn test
```

## Структура проєкту

```
src/
├── main/java/credit/
│   ├── model/          - Моделі даних (Bank, Credit, Client та спеціалізовані типи кредитів)
│   ├── repository/     - Репозиторії для зберігання даних
│   ├── service/        - Бізнес-логіка пошуку та вибору кредитів
│   ├── util/           - Утиліти для роботи з файлами
│   └── Application.java - Головний клас додатку
└── test/java/          - Unit-тести
```

## Функціонал

- **Пошук кредитів** за параметрами клієнта (дохід, кредитний рейтинг)
- **Вибір оптимального кредиту** з урахуванням пріоритетів клієнта
- **Підтримка різних типів кредитів:**
  - Іпотечні кредити (MORTGAGE) - з розрахунком Loan-to-Value
  - Споживчі кредити (CONSUMER) - для особистих потреб
  - Автокредити (CAR) - з розрахунком депреціації
- **Розумний пошук:**
  - За найнижчою процентною ставкою
  - За найкращим рейтингом банку
  - За типом кредиту або банком
  - Гнучкі кредити (дострокове погашення / збільшення ліміту)
- **Розрахунки:**
  - Щомісячний платіж
  - Загальна сума виплат
  - Loan-to-Value для іпотеки
  - Депреціація для автокредитів
- **Збереження даних** у файли JSON (автоматично при виході)
- **Інтерактивний та демонстраційний режими** для зручного використання
