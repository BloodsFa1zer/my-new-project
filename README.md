# Credit System

Консольний додаток для формування пропозицій клієнту по цільових кредитах різних банків.

## Вимоги

- Java 11 або вище
- Maven 3.6+

## Збірка та запуск

```bash
mvn clean compile
mvn exec:java -Dexec.mainClass="ua.edu.credit.Application"
```

## Запуск тестів

```bash
mvn test
```

## Структура проєкту

```
src/
├── main/java/ua/edu/credit/
│   ├── model/          - Моделі даних (Bank, Credit, Client та спеціалізовані типи кредитів)
│   ├── repository/     - Репозиторії для зберігання даних
│   ├── service/        - Бізнес-логіка пошуку та вибору кредитів
│   ├── util/           - Утиліти для роботи з файлами
│   └── Application.java - Головний клас додатку
└── test/java/          - Unit-тести
```

## Функціонал

- Пошук кредитів за параметрами клієнта
- Вибір оптимального кредиту з урахуванням пріоритетів
- Підтримка різних типів кредитів (іпотека, споживчі, автокредити)
- Урахування можливості дострокового погашення та збільшення кредитної лінії
- Збереження даних у файли JSON
